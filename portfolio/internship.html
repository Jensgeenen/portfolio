Technical Overview

  • Backend: Django (Python), PostgreSQL
  • Frontend: Django templates, HTML/CSS
  • Features: Role-based dashboards (Employee, Manager, Admin)
  • Reports: Auto-generated PDFs via ReportLab
  • Deployment: PWA with offline support

Role-Based Functionality

  • Employee: Input water data via form; automatic location detection.
  • Manager: Review submissions; approve and send reports.
  • Admin: Manage users, logs, system monitoring.

What I Learned

I had no prior experience with Django, ReportLab or PWA principles. During the internship, I quickly picked up these tools and successfully applied them in a real-world setting. I also learned how to handle unclear project scopes and language barriers by working independently and structuring my own planning.

Challenges & Solutions
